# pot 


## 安装

安装 `z-pot`

```shell
npm i -g z-pot
```


```

## 使用
```shell
Usage: pot <command> [options]

project overview tool

Options:
  -V, --version  output the version number
  -h, --help     output usage information

Commands:
  report         获取项目概要信息

  Examples:

    $ pot report ## 获取项目概要信息
```
